⚡VOLTAIC

Kendalikan server tiap
cabang toko elektronik Anda.

Voltaic menghubungkan langsung ke akun AWS Anda dan memperlakukan setiap instance EC2 sebagai server sebuah cabang toko — nyala, padam, atau tutup permanen dalam satu dasbor.

  • 01 Sambungkan kredensial AWS Anda
  • 02 Pilih region pusat data cabang
  • 03 Buka, hentikan, atau pantau server cabang

Kredensial hanya disimpan di memori peramban Anda selama sesi berlangsung dan tidak pernah dikirim ke server manapun selain AWS.

SAMBUNGKAN AKUN AWS

Pastikan IAM user memiliki permission EC2 (lihat manual penggunaan untuk contoh policy minimum).
